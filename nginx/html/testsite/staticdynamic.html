This ASP gets the number of seconds from the querystring and simply loops until the time has passed.

... starttime = Hour(Now())*60*60+Minute(Now())*60+Second(Now()) endtime = starttime + Request.QueryString("seconds") Do quittime = Hour(Now())*60*60+Minute(Now())*60+Second(Now()) If (endtime <= quittime) then Exit Do End If Loop ...
